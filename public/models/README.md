# 3D Models

Place your character models here in GLB/GLTF format.

## Recommended Sources

- [Kay Kit](https://kaylousberg.itch.io/) - Low poly modular characters (CC0)
- [Sketchfab](https://sketchfab.com/) - Search "chibi character rigged"
- [Mixamo](https://mixamo.com/) - Auto-rigging and animations

## Expected Files

- `planner.glb` - Planner agent model
- `designer.glb` - Designer agent model
- `coder.glb` - Coder agent model
- `reviewer.glb` - Reviewer agent model

## Animation Clips

Each model should include these animations:
- `idle` - Standing breathing animation
- `walk` - Walking cycle
- `work` - Typing/working gesture
- `wave` - Greeting animation
