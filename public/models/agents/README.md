# Agent Models

Place your agent character models here in GLB/GLTF format.

## Default Agents

- planner.glb — Planner agent (cyan accent)
- designer.glb — Designer agent (magenta accent)
- coder.glb — Coder agent (orange accent)
- reviewer.glb — Reviewer agent (green accent)

## Animation Requirements

| Clip Name | Description |
|-----------|-------------|
| idle | Breathing/standing animation |
| walk | Walking cycle |
| work | Typing/working gesture |

## Recommended Sources

- Kay Kit (kaylousberg.itch.io) — Free low poly characters
- Sketchfab — Search "chibi rigged animated"
- Mixamo — Free rigging and animations
- Meshy — AI generated 3D models
